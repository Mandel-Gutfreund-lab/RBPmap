Versions information

Version 1.2 (Feb. 2021)

  • Added 131 motifs for human RBPs (including 47 new RBPs), that were generated by the RNA bind-n-seq (RBNS) methodology.
    (View RBPmap updated motifs list).
  • The default stringency level was changed to P-value[significant] < 0.01 and P-value[suboptimal] < 0.02.

Version 1.1 (Aug. 2016)

  • Added the latest human genome assembly (GRCh38/hg38).
  • The conservation filter's default has changed to not being applied.
